GLOBAL CYBERSECURITY THREAT ANALYSIS (2015–2024)

Project Overview
This project analyzes global cybersecurity incidents between 2015 and 2024 using Power BI. The dashboard explores the financial impact, affected users, targeted industries, attack types, and exploited vulnerabilities across different countries.

The goal of the analysis is to understand where cyber threats are concentrated, which sectors are most vulnerable, and what security weaknesses attackers most frequently exploit.

Dashboard Questions

This dashboard was designed to answer the following questions:

-What is the total financial loss caused by cyber attacks globally between 2015–2024?
-Which year experienced the highest number of cyber incidents or impact?
-Which country is the worst hit by cyber attacks?
-How many people have been affected by cybersecurity incidents globally?
-Which industries are the most targeted by cyber attacks?
-What attack types cause the highest financial losses?
-Which countries experience the highest number of cyber incidents relative to population?
-What security vulnerabilities are most frequently exploited by attackers?

Dataset Description

The dataset contains simulated cybersecurity incidents with the following fields:

-Country – Country where the cyber attack occurred
-Year – Year of the incident
-Attack Type – Type of cyber attack (Phishing, DDoS, Malware, etc.)
-Target Industry – Industry targeted by attackers
-Financial Loss (in Million $) – Estimated financial damage caused by the attack
-Number of Affected Users – Number of users impacted by the incident
-Attack Source – Source of the attack (Hacker group, Insider, Nation-state, etc.)
-Security Vulnerability Type – Weakness exploited by the attacker
-Defense Mechanism Used – Security mechanism used to mitigate the attack
-Incident Resolution Time (Hours) – Time taken to resolve the incident

Dataset size:
Approximately 1000 cybersecurity incidents from 2015–2024 across multiple countries and industries.

Dashboard KPIs

The dashboard highlights four key indicators:

-Total Financial Loss
-Most Affected Year
-Worst Hit Country
-Total People Affected

These KPIs provide a quick overview of the global cybersecurity landscape.

Key Insights

-Between 2015 and 2024, global cyber attacks resulted in over 151 billion dollars in financial losses.
-Cybersecurity incidents affected more than 1.5 billion users worldwide.
-The United Kingdom recorded the highest impact among analyzed countries.
-IT and Banking sectors were among the most targeted industries.
-Nation-state and unknown attackers contributed significantly to financial losses.

-The most common vulnerabilities exploited include:

--Zero-day vulnerabilities
--Weak passwords
--Social engineering attacks

Tools Used

Power BI-Used for dashboard creation, interactive visualizations, and KPI analysis.

Power Query-Used for data cleaning, column transformation, and data profiling.

DAX-Used for measures and aggregation calculations.

Dashboard Features

-Interactive filters for Country, Year, and Attack Type.
-KPI cards summarizing major cybersecurity indicators.
-Industry-level analysis of cyber attack targets.
-Attack type analysis based on financial losses.
-Vulnerability analysis highlighting common security weaknesses.
-Summary insights describing key findings from the data.

Project Purpose

This project demonstrates how Power BI can be used to analyze cybersecurity trends, identify high-risk industries, understand attack patterns, and support better cybersecurity decision-making.
